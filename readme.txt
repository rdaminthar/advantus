1.Advantus Application developed from Laravel and Vue JS
2.Project Scaffolding is done using Laravel Breeze
3.Moment dependancy library is used to format the date and Jquery and bootstrap is used for styling purposes. Repository pattern is used where necessary
4.Mix is used for webpack css and javascript
 
How to setup the project:
1.Clone the repository (master branch)
2.Run 'composer update' command
3.Run 'php artisan migrate' to migrate the database tables
4.Run 'php artisan db:seed CitySeeder' command to seed the cities
5.Run 'npm run hot' command (since css and js files are called using mix)
